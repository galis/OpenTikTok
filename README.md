# 开源视频编辑项目

>初衷:就是想模仿一个类似剪映编辑的项目，自己在音视频干了一段时间，想沉淀一下。
raw目录下有testfile.mp4测试文件，可以push到/data/data/com.galix.opentiktok/cache测试.
目前基于testfile.mp4的两段视频，功能基本验证完毕，后续持续修复bug.

##感谢到大家对项目认可，项目在8月份会持续更新，和大家一起进步。
